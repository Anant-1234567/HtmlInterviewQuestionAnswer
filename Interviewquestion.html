1)Difference between div and span?

Ans:- Div- Div is a Block level element,div is bound to take whole width of the screen,div is taking full space of line.
Span- Span is a inline element,span don't take full width of the screen,it takes only text spaces.

2)What is difference between class and id's?

Ans:- Class- Class is not unique and we can use same class in multiple element,Multiple class selector can be attaches to an element.
ID- id is unique and each element can have one id,each page can have one id ,only one id selector can be attached to an element.

5)Difference between block and inline?

Ans:- Block level element- Block level element always start in a new line,Block element occupy the full width irrespective of their sufficiency,Block element have top and bottom marfgin.
Inline Element- Inline element don't start in a new line,Inline element occupy only sufficient width required,Inline element don't have top and bottom margin.

6)Difference between em and strong?

Ans:- em tag- the em tag should be used to represent stress emphasis, like when you would read the emphasized text in a different tone of voice.
Strong tag- the strong tag should be used to indicate strong importance,seriousness,or urgency ,like to indicate key phrases in a text for some one skimming it.

7)What are attributes?

Ans:- HTML attributes are special words which provide additional information about the elements or attributes are the modifier of the HTML element.
Each element or tag can have attributes, which defines the behaviour of that element.
Attributes should always be applied with start tag.
The Attribute should always be applied with its name and value pair.
The Attributes name and values are case sensitive, and it is recommended by W3C that it should be written in Lowercase only.
You can add multiple attributes in one HTML element, but need to give space between two attributes.

8)Difference between HTML tags and elements?

Ans:- HTML tags- HTML tags are used to hold the HTML elements,HTML tags start with <(opening tag)and end with >(closing tag).
HTML element- HTML element hold the content,whatever written within a HTML tag are HTML element.
